# BEAPER Nano Beginner Activities

These beginner programming activities are designed to introduce novices to
microcontroller programming using MicroPython running on the Arduino Nano ESP32
microcontroller. These activities are created specifically for use with the
BEAPER nano circuit hardware -- feel free to modify them for use with other
hardware devices.

## Curriculum Overview

- the Microcontroller Core Curriculum file provides a complete curriculum outline
